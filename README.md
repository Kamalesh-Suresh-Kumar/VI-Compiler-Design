# Compiler Design Laboratory
## Semister: VI

## Repository Structure

| Lab No | Title                                      | Description |
|--------|-------------------------------------------|-------------|
| LAB1   | String Manipulation Programs in C         | Implementation of basic string operations such as character count, length calculation, reverse, copy, removing spaces, character frequency, and vowel/consonant counting. |
| LAB2   | Lexical Analyzer using C                  | Development of a simple lexical analyzer in C to identify tokens such as keywords, identifiers, operators, delimiters, and special symbols from input code. |
| LAB3   | Arithmetic Expression Recognition using LEX | Implementation of a LEX program to recognize valid arithmetic expressions containing integers as identifiers and operators (+, *) and to count them separately. |
| LAB4   | Comment Removal using LEX                 | Implementation of a LEX program to remove single-line (`//`) and multi-line (`/* */`) comments from a C source file and generate a clean output file while preserving formatting and structure. |
| LAB5   | AⁿBⁿ Recognition using YACC              | Design of a YACC parser to recognize strings of the form AⁿBⁿ (equal number of A’s followed by B’s) and validate them using context-free grammar rules. |
| LAB6   | Simple Calculator using YACC             | Development of a calculator using YACC to evaluate arithmetic expressions with operators such as addition, subtraction, multiplication, and division with proper precedence handling. |
| LAB7   | Syntax Tree Generation using YACC        | Implementation of a YACC parser to construct and display an Abstract Syntax Tree (AST) for given expressions, demonstrating parsing and tree generation concepts. |
